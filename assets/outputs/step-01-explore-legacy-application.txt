7. KEY FINDINGS
────────────────────────────────────────────────────────────────────────────────

1. QuotingEngine.cs is the HEART of the system (685 lines, 23.5% of codebase)
   - 15-step premium calculation pipeline
   - 40+ rating factors
   - 50 state-specific compliance rules
   - Nested if/else chains for state variations (CA, FL, TX, NY, LA)

2. Business logic is tightly coupled:
   - QuotingEngine instantiates PremiumCalculator and ComplianceService directly
   - No dependency injection — all services use `new` keyword
   - Domain rules classes reference each other

3. Integration points use legacy protocols:
   - WCF basicHttpBinding for reinsurance (transport security)
   - MSMQ paths hardcoded in Web.config
   - Regulatory reporting via HTTP/HTTPS endpoints

4. Cannot build or run without:
   - .NET Framework 4.6.1 Developer Pack
   - SQL Server LocalDB
   - IIS Express
   - Visual Studio with ASP.NET MVC 4 workload
